## 2025-03-25

### 项目结构

- 后端逻辑集中到 Python 包 **`paper_rewrite/`**（路径、配置、分段与任务、LLM 调用、Word 提取），根目录保留 **`server.py`** 作为 FastAPI 入口，便于启动与 PyInstaller 打包。
- 测试与辅助脚本放在 **`tests/`**：`test_smoke.py`（pytest）、`qa_smoke.py`（手工冒烟）、`integration_llm_env.py`（需环境变量 API Key 的端到端联调）。

### 功能与行为

- **数据预处理**：页面支持上传 `.docx`，提取正文纯文本写入 `论文.txt`（接口 `POST /api/upload-docx`）。
- **启动前校验**：`POST /api/start` 在返回前校验 API Key 与可读提示词，避免「已返回 job_id 但后台因配置失败」。
- **HTTP 客户端**：进程内复用 `httpx.AsyncClient`（lifespan），并限制连接池。
- **任务与内存**：内存中仅保留最近约 **20** 个任务；更早的 `job_id` 将失效。
- **配置扩展**：`default.yaml` 支持 `temperature`、`request_timeout_sec`、`max_tokens`、`anthropic_max_tokens` 等生成参数。
- **LLM 响应**：对各厂商返回 JSON 做更稳健的解析与错误信息。
- **其它**：`POST /api/frontend-log` 限制单条体积；SSE 流式相关日志降为 DEBUG 级别。

### 依赖与测试

- `requirements.txt` 含 `pytest`（冒烟用）。
- 推荐命令：在项目根目录执行 `pytest`；手工冒烟：`python tests/qa_smoke.py`。
